# Treadly

The most awesome AI-powered online t-shirt shop.

## Introduction

You're are building Threadly, a small but realistic online t-shirt shop. The shop sells customizable t-shirts and is operated by a small company with limited engineering capacity. Speed, quality, and maintainability matter.

The company wants to:
- Launch fast
- Continuously improve the product
- Use AI to amplify a small team
- Prepare for future scale without overengineering

The case is deliberately incomplete at the start. Requirements, constraints, and quality expectations increase throughout the day. 

You can work in any reasonable stack (web app + simple backend), but AI is treated as a core development tool, not an add-on.

## Prerequisites

- Standard internet browser
- Dev IDE tool (vs code / visual studio  etc.)
- AI tool (preferable with API access)
  - ChatGPT Codex
  - Claude Code
  - Github Copilot (free tier is possible)
- Software
  - Node.js 18 or newer
  - Python 3.10+
- Access to an LLM via e.g., Microsoft Foundry, OpenAI, Claude, etc.
