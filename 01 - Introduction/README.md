# Introduction

## Setting up the solution

make sure to start docker desktop

```csharp
// Open the case solution folder
cd threadly-app
```

```csharp
// Make sure that all npm modules are installed
cd threadly-app/src/webapp
npm install
```

```csharp
// Run the webapp
cd threadly-app/src/webapp
npm run dev
// open browser http://localhost:7608/
// and enjoy a 5-star monochrome webshop
```

```csharp
// Run the .NET Aspire solution
cd threadly-app
aspire run
// open browser https://localhost:7601/
```
