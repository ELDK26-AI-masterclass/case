# Introduction

## Setting up the solution

make sure to start docker desktop

```csharp
// Open the src folder
cd code
```

```csharp
// Make sure that all npm modules are installed
cd code/src/frontend
npm install
```

```csharp
// Run the web application alone
cd code/src/frontend
npm run dev
```

```csharp
// Run the .NET Aspire solution
cd code
aspire run
```
