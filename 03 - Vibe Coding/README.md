# Vibe Coding

## Workshop Goal

In this workshop, you will:

- Use an AI coding platform (such as GitHub Spark, Cursor, Figma Make, or similar) to build a simple t-shirt shop web app.
- Practice structured prompting to guide implementation.
- Iterate on features through refinement prompts.
- Download/export the generated project and run it locally.

## Case Description

Build a simple t-shirt shop application with:

- A landing page
- A product list with 3–5 t-shirts
- A product detail view
- A front-end-only "Add to cart" confirmation (no real checkout)

## Step-by-Step Instructions

### Step 1 – Choose an AI coding platform

- Select one platform: GitHub Spark, Cursor, Figma Make, or another AI coding tool.
- Create a new project/workspace for the t-shirt shop case.

### Step 2 – Write a structured prompt to build the t-shirt shop

Participants should create their own initial prompt.

Use the **Case Description** above as your product brief, then write a prompt with this structure:

- Intent: what you are building and for whom.
- Flow: how users move through the experience.
- Scope: what to include and what to exclude.
- Output format: what files/components you want the tool to generate.

Keep it concise and specific.

### Step 3 – Add at least one refinement prompt

- Submit at least one follow-up refinement prompt, for example:
	- Improve spacing, typography, and visual hierarchy.
	- Add a cart item counter in the top navigation.
	- Improve mobile responsiveness for product cards.

### Step 4 – Download/export the generated project

- Use the platform’s export/download option.
- Save the project locally and open it in your editor.

### Step 5 – Run the project locally

Use a typical Node/Vite-style workflow:

```bash
npm install
npm run dev
```

- Open the local URL shown in the terminal (for example, `http://localhost:5173`).

## Prompt Guidance

When prompting, keep instructions short and structured:

- Be clear about intent.
- Define the user flow.
- Specify scope positively (what to include and what not to include).
- Keep requirements organized with bullets.

## What This Exercise Demonstrates

- AI can significantly accelerate prototyping.
- Output quality depends on how clearly you direct the tool.
- Vibe coding is collaborative and iterative: prompt, review, refine.
