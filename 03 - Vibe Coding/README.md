# Vibe Coding

In this step, you will:
- Use an vibe coding platform to build the initial content of Threadly.
- Download/export the generated project and run it locally.

## Getting vibe coding tool up and running
- Launch your favorite vibe coding tool.
- Create a new project called Threadly.
- Use an e-commerce or online shop template if available.

## Building the online shop
- Create a landing page.
- Create a product list with 3–5 t-shirts.
- Create a simple product detail view.
- Create a fake “add to cart” experience (no real checkout yet).

## Download/export the generated project
- Use the export/download option.
- Save the project locally and open it in your editor.

## Run the project locally
Use a typical Node/Vite-style workflow:

```bash
npm install
npm run dev
```

- Open the local URL shown in the terminal (for example, `http://localhost:5173`).
