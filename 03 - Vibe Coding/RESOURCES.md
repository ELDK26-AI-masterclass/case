## Troubleshooting
- If `npm` is not recognized, install Node.js LTS: https://nodejs.org/
- If dependency install fails, delete `node_modules` and run `npm install` again.
- Typical run commands:
	- `npm install`
	- `npm run dev`
- If the default port is in use, run: `npm run dev -- --port 5174`
