# Spec-Driven Development

In this step, you will:
- Use spec-driven development to orchestrate changes to your online shop.

## Open the online shop code
Open the solution in your preferred AI tool.

## Install spec engine
Install OpenSpec on your computer
```bash
npm install -g @fission-ai/openspec@latest
```

## Initialize spec engine
Initialize OpenSpec in your solution
```bash
cd your-project
openspec init
```

## Add dark mode
Add dark mode as a new feature using step-by-step change.
Remember to archive the change when it is done.

## Create second change
Add multi-language as a new feature using fast-forward change.
Remember to archive the change when it is done.
