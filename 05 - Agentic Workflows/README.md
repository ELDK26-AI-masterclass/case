# Agentic Workflows

In this step, you will:

- Create a custom agent.
- Create instruction files.
- Create prompt files.

## Open the online shop code

- Open the solution in your preferred AI tool.

## Create a custom agent

- Create a custom agent with clear boundaries on what it can and cannot do.
- Define the agent's capabilities and limitations to ensure it operates within the desired scope.

## Create instruction files

- Create instruction files that provide clear and concise guidelines for the agent's behavior and decision-making processes.
- Create instruction files that work on specific areas of the project, such as frontend, backend etc.

## Create prompt files

- Create reusable prompt files for repeated tasks that you want the agent to perform. These prompts should be designed to elicit specific responses from the agent, making it easier to achieve consistent results.
