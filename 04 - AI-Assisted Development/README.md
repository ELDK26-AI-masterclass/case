# AI-Assisted Development

In this step, you will:
- Use AI agents locally and remote.
- Use AI agents interactive and non-interactive.

## Open the online shop code
- Open the solution in your preferred AI tool.

## Make change locally interactive
- Ensure you have selected locally in chat interface.
- Make change to the solution using chat interface.

## Make change locally non-interactive
- Ensure you have selected background in chat interface.
- Make change to the solution using chat interface.

## Make change remote non-interactive
- Ensure you have selected cloud in chat interface.
- Make change to the solution using chat interface.
